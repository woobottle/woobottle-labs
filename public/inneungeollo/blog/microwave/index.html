가스레인지 없어도 OK — 자취 전자레인지 요리 5가지

2026-07-01

원룸·기숙사라 가스레인지가 없거나 설거지가 귀찮을 때, 전자레인지 하나로 되는 요리가 진짜 구원이에요. 그릇 하나에 재료 넣고 돌리기만 하면 끝나는 자취 전자레인지 요리 5가지를 모았어요.

1. 전자레인지 계란찜 (5분 · 쉬움)

머그컵 하나로 되는 폭신 계란찜.

재료: 계란 2개, 물(또는 우유) 반 컵 / 소금, 대파 약간

만드는 법 1. 큰 머그컵에 계란을 풀고 물·소금을 섞어요. 2. 송송 썬 대파를 넣어요. 3. 랩을 살짝 씌우거나 뚜껑을 덮어요. 4. 전자레인지에 2분 → 저어주고 → 1분 더 돌려요.

팁: 한 번에 오래 돌리면 부풀어 터져요. 나눠 돌리세요.

2. 전자레인지 감자버터 (8분 · 쉬움)

간식·안주로 딱. 감자 하나면 돼요.

재료: 감자 1개, 버터 / 소금, 후추

만드는 법 1. 감자를 씻어 포크로 여기저기 찔러요. 2. 젖은 키친타월로 감싸 전자레인지에 5분. 3. 뒤집어 2~3분 더 돌려 속까지 익혀요. 4. 반 갈라 버터·소금·후추를 올려요.

팁: 포크로 찔러 안 걸리면 다 익은 거예요.

3. 전자레인지 라면 (6분 · 쉬움)

불 없이 끓이는 라면. 그릇 하나면 끝.

재료: 라면, 물 550ml

만드는 법 1. 전자레인지 용기에 면·스프·물을 넣어요. 2. 뚜껑을 살짝 열고 5분 돌려요. 3. 계란을 풀어 넣고 1분 더 돌려요.

팁: 물이 넘칠 수 있으니 넉넉한 그릇에, 뚜껑은 꼭 살짝 열어두세요.

4. 전자레인지 두부계란 (7분 · 쉬움)

단백질 든든한 초간단 한 그릇.

재료: 두부 반 모, 계란 1개 / 간장, 참기름, 대파

만드는 법 1. 두부를 으깨 그릇에 담아요. 2. 계란을 풀어 섞고 간장·대파를 넣어요. 3. 전자레인지에 2분 → 저어주고 → 1분 더. 4. 참기름을 둘러 마무리해요.

팁: 두부 물기를 빼면 더 꾸덕하게 나와요.

5. 전자레인지 김치볶음밥 (8분 · 쉬움)

팬 없이도 되는 김치볶음밥.

재료: 밥, 김치, 참치캔(선택) / 참기름, 김가루

만드는 법 1. 그릇에 밥·잘게 썬 김치·참치를 넣고 섞어요. 2. 전자레인지에 3분 돌려요. 3. 꺼내서 잘 섞고 1~2분 더 돌려요. 4. 참기름·김가루를 뿌려요.

팁: 마지막에 계란프라이(이것도 전자레인지 가능!)를 올리면 완성도가 확 올라요.
